# Screenshots Directory

This directory contains screenshots of the MatchMind application.

## Required Screenshots

1. **main-interface.png** - Full application view showing teams, form, and prediction sections
2. **team-form.png** - Team selection with form display showing match results
3. **prediction.png** - Prediction interface with probability bars and feature explanations
4. **api-docs.png** - Swagger UI documentation page

## How to Capture Screenshots

1. Start the application:
   ```bash
   docker compose up
   ```

2. Open http://localhost:5173 in your browser

3. Capture screenshots:
   - **Main Interface**: Full page screenshot of the initial view
   - **Team Form**: Select a team and capture the form section
   - **Prediction**: Make a prediction and capture the results
   - **API Docs**: Navigate to http://localhost:8000/docs and capture

4. Save screenshots with the exact filenames listed above

## Image Guidelines

- **Format**: PNG or JPG
- **Size**: Recommended 1920x1080 or similar aspect ratio
- **Quality**: High resolution, clear text
- **Browser**: Use Chrome or Firefox for consistent rendering
